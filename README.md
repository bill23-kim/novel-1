## About

**컴파일 되지 않는 마음**은 카카오의 시니어 백엔드 개발자 **김빌**과 건축 인테리어 디자이너 **서하윤**의 이야기입니다.

코드로 세상을 이해하는 남자와, 공간으로 사람을 읽는 여자.
서로 다른 언어를 가진 두 사람이 판교의 한 카페에서 만나
사랑이라는, 컴파일되지 않는 감정 앞에 서게 됩니다.

```
     장르  :  현대 로맨틱 소설
     분량  :  장편 20챕터 (약 100,000자)
     배경  :  2026년 봄 ~ 2027년 초봄, 서울/판교
     키워드 :  개발자 로맨스, 코드와 공간, 오픈소스, 판교, 카카오
```

## Chapters

모든 챕터 제목은 개발 용어로 구성되어 있습니다.
코드의 생애주기가 곧 사랑의 생애주기입니다.

### Act 1 — 만남과 설렘

```
  ┌─────────────────────────────────────────────────────┐
  │  Ch 1   init             초기화                      │
  │  Ch 2   handshake        연결 수립                    │
  │  Ch 3   ping             신호 보내기                  │
  │  Ch 4   dependency       의존성 추가                  │
  │  Ch 5   first commit     첫 번째 커밋                 │
  └─────────────────────────────────────────────────────┘
```
> 판교 카페에서의 우연한 만남. 사옥 리뉴얼 프로젝트로 엮인 인연.
> 코드와 공간, 서로 다른 언어가 처음으로 교차하는 봄.

### Act 2 — 가까워짐

```
  ┌─────────────────────────────────────────────────────┐
  │  Ch 6   merge conflict   병합 충돌                    │
  │  Ch 7   refactor         리팩토링                     │
  │  Ch 8   push             푸시                        │
  │  Ch 9   sync             동기화                      │
  │  Ch 10  deploy           배포                        │
  └─────────────────────────────────────────────────────┘
> 전 여자친구의 등장, 오해와 해명, 질투와 확신.
> 그리고 판교 야경 아래, 처음으로 마음을 배포하는 밤.

### Act 3 — 갈등과 위기

```
  ┌─────────────────────────────────────────────────────┐
  │  Ch 11  exception        예외 발생                    │
  │  Ch 12  timeout          시간 초과                    │
  │  Ch 13  deadlock         교착 상태                    │
  │  Ch 14  crash            크래시                      │
  │  Ch 15  disconnect       연결 끊김                    │
  └─────────────────────────────────────────────────────┘
```
> 현실이 코드처럼 깔끔하지 않다는 것을 깨닫는 계절.
> 커리어와 사랑 사이, 선택할 수 없는 선택 앞에 서다.

### Act 4 — 해소와 성장

```
  ┌─────────────────────────────────────────────────────┐
  │  Ch 16  rollback         롤백                        │
  │  Ch 17  hotfix           긴급 수정                    │
  │  Ch 18  async await      비동기 대기                   │
  │  Ch 19  reconnect        재연결                      │
  └─────────────────────────────────────────────────────┘
```
> 가장 비싼 롤백. 가장 긴 기다림.
> 그리고 공간이 전하는 고백을 읽어내는 가을.

### Act 5 — 결말

```
  ┌─────────────────────────────────────────────────────┐
  │  Ch 20  heartbeat        심장 박동                    │
  └─────────────────────────────────────────────────────┘
```
> `feat: add heartbeat monitoring for the one who matters most`
