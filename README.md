## About

**컴파일 되지 않는 마음**은 카카오의 시니어 백엔드 개발자 **김빌**과 건축 인테리어 디자이너 **서하윤**의 이야기입니다.

코드로 세상을 이해하는 남자와, 공간으로 사람을 읽는 여자.
서로 다른 언어를 가진 두 사람이 판교의 한 카페에서 만나
사랑이라는, 컴파일되지 않는 감정 앞에 서게 됩니다.

| | |
|---|---|
| **장르** | 현대 로맨틱 소설 |
| **분량** | 장편 20챕터 (약 100,000자) |
| **배경** | 2026년 봄 ~ 2027년 초봄, 서울/판교 |
| **키워드** | 개발자 로맨스, 코드와 공간, 오픈소스, 판교, 카카오 |
