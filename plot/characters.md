# 캐릭터 프로필 & 관계도

## 주인공

### 김빌 (Kim Bill) — 남주인공

**기본 정보**
- 나이: 33세
- 직업: 카카오(Kakao Corporation) 메시징 플랫폼팀 시니어 백엔드 개발자
- 거주지: 서울 성수동 원룸 (판교까지 셔틀버스 통근)
- GitHub: bill23-kim
- 오픈소스 활동: scouter-project 핵심 멤버 (APM 모니터링 도구)
- 업적: Arctic Code Vault Contributor, Pull Shark x2

**외모**
- 175cm, 마른 체형. 어깨가 약간 굽어 있다 (오래 앉아서 코딩한 흔적)
- 짧고 깔끔하게 자른 머리, 가끔 며칠간 못 자를 때 앞머리가 눈을 찌름
- 검은 뿔테 안경. 안경 너머로 보는 눈이 의외로 깊고 진지하다
- 평상시 카카오 로고가 작게 새겨진 후드티 + 청바지 + 뉴발란스 990
- 손가락이 길고 마디가 뚜렷함 (키보드 위에서 가장 빛나는 손)
- 웃을 때 한쪽 볼에만 보조개가 생기는데 본인은 모름

**성격**
- 코드 앞에서는 완벽주의, 사람 앞에서는 서투른 타입
- 말수가 적지만 한번 입을 열면 논리적이고 핵심을 찌름
- 감정 표현이 서툴러서 중요한 말을 텍스트(카톡/슬랙)로 먼저 꺼내는 습관
- 문제를 만나면 무조건 근본 원인(root cause)부터 파고드는 사고방식
- 의외로 고집이 세고 자기 코드에 대한 자부심이 강함
- 하지만 좋은 코드 리뷰 피드백에는 즉시 반영하는 유연함도 있음
- 새벽 2~3시가 가장 집중이 잘 되는 시간. 이때 가장 중요한 커밋을 함

**습관 & 디테일**
- 생각할 때 무의식적으로 키보드 타이핑하는 손동작을 함 (에어 코딩)
- 카페에서 항상 콘센트 옆자리에 앉음
- 커피는 아메리카노 only, 얼음 적게 ("메모리처럼 여유 공간이 있어야 해")
- 핸드폰 알림은 GitHub, Slack, 서버 모니터링 알림만 켜둠
- 감정이 복잡해지면 산책 대신 코드를 짬 (감정의 디버깅)
- 좋아하는 사람에게 관심을 표현하는 방식: 상대의 패턴을 관찰하고 기억함

**매력 포인트**
- 자기 분야에 대한 순수한 열정을 이야기할 때 눈이 반짝임
- 묵묵히 문제를 해결해주는 사람. 말보다 행동이 앞섬
- scouter 프로젝트 글로벌 컨트리뷰터들과 영어로 소통하는 모습의 갭
- 서투른 고백, 어색한 다정함이 주는 진정성

## 여주인공

### 서하윤 (Seo Hayun) — 상대역

**기본 정보**
- 나이: 31세
- 직업: 건축 인테리어 디자인 스튜디오 '온기(ONGI)' 대표 디자이너 겸 공동 창업자
- 거주지: 서울 연남동 (작업실 겸 자택)
- 학력: 홍익대학교 실내건축학과 졸업, 덴마크 코펜하겐 교환학생 (1년)

**외모**
- 167cm, 날씬하지만 건강한 체형 (현장 돌아다녀서 체력 좋음)
- 어깨 아래로 내려오는 웨이브 머리, 작업할 때는 항상 연필로 올려 꽂음
- 화장 거의 안 하지만 입술에 색감 하나는 꼭 넣는 스타일
- 린넨 셔츠, 와이드 팬츠, 가죽 토트백 — 깔끔하면서 감각적인 차림
- 손톱에 항상 물감이나 마커 자국이 남아 있음
- 눈이 크고 표정이 풍부해서 감정이 얼굴에 바로 드러남

**성격**
- 공간을 읽는 감각이 탁월함. 어떤 장소에 들어가면 그곳의 '결'을 직감적으로 파악
- 따뜻하고 사교적이지만 자기 일에 대해서는 단호하고 타협하지 않음
- 감정에 솔직한 편. 좋으면 좋다, 싫으면 싫다고 말하는 직구 타입
- 하지만 진짜 중요한 감정(사랑)에 대해서는 의외로 신중하고 두려움이 있음
- 과거 연인에게 "넌 너무 자기 세계에 빠져 산다"는 말을 들은 상처가 있음
- 사람을 만나면 그 사람이 어떤 '공간'에 어울리는지를 상상하는 습관

**습관 & 디테일**
- 항상 스케치북과 6B 연필을 가지고 다님
- 카페에서 주변 사람들의 동선과 자세를 관찰하며 메모
- 커피보다 차를 좋아함 (캐모마일 라테)
- 작업에 몰두하면 시간 감각을 완전히 잃어버림
- 스트레스받으면 가구를 재배치함 (자기 방이든 사무실이든)
- 건물 지나갈 때 무의식적으로 천장 높이를 가늠함

**주인공과의 접점**
- 카카오 판교 사옥 리뉴얼 프로젝트 디자인 총괄로 수주
- 판교 알파돔시티 카페에서 우연히 만남 (같은 단골 카페)
- 개발자들의 업무 환경을 이해하기 위해 빌의 팀을 관찰/인터뷰하게 됨

**매력 포인트**
- 빌의 코드 세계를 자기만의 언어(공간, 구조, 빛)로 번역해서 이해하려는 노력
- 감정 표현이 서투른 빌에게 공간과 디자인으로 마음을 전하는 방식
- 당당하면서도 연약한 면이 공존하는 입체적 캐릭터
